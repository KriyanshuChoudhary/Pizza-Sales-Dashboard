# Pizza-Sales-Dashboard

Interactive Pizza Sales Dashboard built using SQL Server and Power BI to analyze key business metrics — including revenue, orders, and product performance.
Features include KPI tracking, trend analysis, and identification of top & bottom-selling pizzas for data-driven insights.

---

## Project Overview
The Pizza Sales Dashboard is an end-to-end data analysis project that tracks and visualizes key business metrics such as total revenue, order trends, product performance, and category sales.
It provides actionable insights for decision-making by identifying the best and worst-performing pizzas, sales trends across time, and customer purchase behavior.

---


## Objectives
- Analyze pizza sales data to uncover performance trends.
- Identify top and bottom-selling pizzas by revenue, quantity, and orders.
- Understand sales distribution across categories and sizes.
- Build an interactive and visually appealing dashboard using Power BI.

--- 

## Key Insights
- Total Revenue: ₹817.86K
- Total Orders: 21,350
- Total Pizzas Sold: 49.57K
- Average Order Value: ₹38.31
- Top-Selling Pizza: The Thai Chicken Pizza
- Most Popular Size: Large (45.89% of sales)
- Peak Sales Day: Friday
- Highest Sales Month: July
